# GYDS Blockchain

GYDS is a custom blockchain with native coins GYDS and GYD, CPU mining, liquidity pools, and DeFi features.

## Features
- Blockchain with blocks, transactions, and mining
- HD Wallets with seed phrases
- Native tokens: GYDS (main) & GYD (secondary)
- User-generated tokens via off-chain service
- Liquidity pools for GYDS↔GYD and GYD↔USDT swaps
- RPC endpoints for web/mobile interaction
- Private P2P network via Kerio VPN
